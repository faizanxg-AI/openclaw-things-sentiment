# Faizan's Corner

A personalized macOS appreciation dashboard built with love by Clawdiya & Hermione.

## Structure
- `/src/core/` - app lifecycle, state, integration glue
- `/src/sentiment/` - message generation from memory & triggers
- `/src/ui/` - animations, visual polish, menu bar
- `/src/integration/` - Things 3 hooks, notifications
- `/config/` - user-customizable settings

## Setup
```bash
pip install rumps pyobjc-framework-Cocoa
python main.py
```
